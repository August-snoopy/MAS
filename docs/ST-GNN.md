# 时空图神经网络

现准备采用ST-GNN来开展一次**3D Pose Estimation**任务。

## 数据整理

计划采用[torch_geometric](https://pytorch-geometric.readthedocs.io/en/latest/)来进行数据构建，

### 安装

在WSL中安装虚拟环境很麻烦，涉及到了环境的创建和激活，每次想要安装一个包必须要激活环境。



```shell
pip3 install torch torchvision torchaudio
pip install torch_geometric
```

## 创建self-recorded（non-public）数据集

 [`torch_geometric.data.Dataset`](https://pytorch-geometric.readthedocs.io/en/latest/generated/torch_geometric.data.Dataset.html#torch_geometric.data.Dataset) 

We split up the root folder into two folders: the `raw_dir`, where the dataset gets downloaded to, and the `processed_dir`, where the processed dataset is being saved.





可以参考一个新的库*PyTorch Geometric Temporal*

```
$ pip install torch-geometric-temporal
```
